# Data Clustering

## Build (Command Line)

```bash
cmake -S . -B build
cmake --build build
```

Run: `./build/data_clustering <input_file> <clusters> <max_iter> <threshold> <runs>`

## VS Code

1. Install "CMake Tools" extension
2. Open folder, run "CMake: Configure" from command palette
3. Press F7 to build

## Visual Studio

1. File → Open → Folder
2. Build → Build All
